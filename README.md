# Statistics in SQL

## Project Overview
This project demonstrates how statistical analysis can be performed using Python. 
The notebook explores descriptive statistics and statistical concepts using a dataset of height, weight, and BMI information.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- ## Key Concepts Covered
- Descriptive Statistics
- Mean, Median, Mode
- Data Visualization
- Distribution Analysis
